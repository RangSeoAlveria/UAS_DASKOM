Deskripsi Proyek

Website kasir ini adalah proyek berbasis web yang saya kembangkan menggunakan PHP, MySQL, dan Bootstrap. Tujuannya adalah untuk memudahkan pencatatan transaksi dan manajemen laporan penjualan secara digital di toko atau usaha kecil menengah.

Fitur Utama

  • Login admin dan keamanan sederhana
  • Pencatatan transaksi penjualan
  • Manajemen data barang dan stok
  • Pencetakan struk
  • Export laporan penjualan

Teknologi yang Digunakan

  • PHP Native
  • MySQL Database
  • Bootstrap 5 untuk tampilan
  • VSCode dan Laragon sebagai tools development

Catatan: Dibuat sebagai bagian dari ujian praktik/kompetensi di kelas untuk menunjukkan pemahaman dasar pemrograman web dan logika transaksi.
