Beranda Tentang Saya Proyek

Tentang Proyek

Ini adalah proyek tugas kelas 11 dengan tema bebas. Saya memilih membuat website bertema *Call of Duty : Modern Warfare* karena saya suka dengan alur cerita dan gameplay-nya yang seru. Website ini terdiri dari beberapa menu utama yaitu Story Game, Gallery, dan About Me.

Fitur Website

Story Game

Berisi ringkasan cerita utama dari COD : MW, lengkap dengan karakter penting dan latar tempat yang digunakan dalam game.

Gallery

Menampilkan gambar-gambar menarik dari dalam game seperti momen aksi, karakter ikonik, dan cuplikan sinematik.

About Me

Halaman ini menjelaskan siapa saya, alasan saya memilih tema ini, dan teknologi apa yang saya gunakan untuk membuat web ini.

Catatan Proyek

Proyek ini dibuat menggunakan HTML, CSS, dan Bootstrap sebagai bagian dari tugas akhir semester. Saya belajar bagaimana membuat struktur halaman, menata tampilan dengan CSS, dan menyusun layout yang menarik menggunakan Bootstrap. Web ini menjadi salah satu pengalaman awal saya membangun website dari nol.
